## 🚀 Sobre o Projeto

Este é o repositório oficial do meu portfólio profissional. Ele foi concebido sob uma ótica editorial e minimalista, focando em entregar uma experiência de alto nível (high-end) para parceiros de negócios e recrutadores técnicos. 

O projeto vai além de uma simples vitrine; é uma demonstração de engenharia de software aplicada ao design e à performance.

### ✨ Diferenciais Técnicos & Criativos

- **🎭 Design Editorial Assimétrico**: Uma composição moderna que foge do óbvio, inspirada em publicações de luxo e marcas de tecnologia elite.
- **🍱 Bento Grid Moderno**: Organização visual estratégica de serviços e competências usando a tendência de grids da Apple/Vercel.
- **⚡ Performance Extrema**: Otimização completa de carregamento, fontes locais e imagens para garantir o Core Web Vitals no verde.
- **📈 SEO de Classe Mundial**: Configuração completa de metadados, OpenGraph dinâmico com Next.js e cartões sociais personalizados.
- **⚓ Floating Dock Navigation**: Uma experiência de navegação inspirada em sistemas operacionais modernos, reagindo ao scroll do usuário.

## 🛠️ Stack Tecnológica

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Estilização:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animações:** [Framer Motion](https://www.framer.com/motion/)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **SEO/Metadata:** APIs nativas do Next.js (sitemap, robots, manifest)

## 📦 Começando

### Pré-requisitos
- Node.js 20+
- npm / yarn / pnpm

### Instalação

```bash
# Clone o repositório
git clone git@github.com:Allysonfreitas210695/Portifolio-profissional.git

# Acesse a pasta do projeto
cd portfolio-web

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## 🎨 Filosofia de Design

O design deste portfólio segue uma filosofia de **Curadoria Digital**. Cada pixel, efeito de borrão (`backdrop-blur`) e animação de revelação (`stagger reveal`) foi planejado para guiar o usuário de forma suave, transmitindo confiança e atenção aos detalhes — qualidades essenciais em projetos B2B de grande escala.
